支持文章管

导出成 docx (Oce Word 文档), PDF、HTML 网页、TXT 以及图片格式。而且作为国产软件，HTML 网页 除了支持导出 WordPress 兼容格式外，还能支持微信公众号的格式。对微信排版个性化程度不高，可 以排版后放到秀米等美化一下。

3、Typora \ 免费极简 Markdown 编辑器！

Typora 抛弃了传统的交互模式，它只使用一个窗口，却能优雅地实现同时将代码编辑与预览结合起 来！从而为用户带来更加流畅直观的，所见即所得的 Markdown 写作体验。

Android 安卓平台上优秀的 Markdown 编辑器却很少！所以今天这款免费的 MarkdownX 真的值得推 荐…… 它能满足我们随时记笔记和写作的需求。
